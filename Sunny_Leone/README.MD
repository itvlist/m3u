珊妮·粱妮 (Sunny Leone)
珊妮·里昂（英语：Sunny Leone，1981年5月13日－），为印度裔加拿大人，是一位著名的美国、加拿大和印度三重国籍的前色情演员 、宝莱坞演员及成人模特儿。
她也有使用艺名 Karen Malhotra。2003年她曾登上《阁楼杂志》的封面，并在当年被评为该杂志的‘年度阁楼宠物’，也是生动娱乐的签约表演者，更被美信命为2010年12大成人影星。
自2011年起，珊妮已经嫁给了色情演员同事丹尼尔·韦伯。
